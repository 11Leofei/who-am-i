# 星尘 · 身份 — Stardust Identity

> 在浩瀚宇宙中，寻找你独特的频率。

一款宇宙主题的沉浸式人格探索 Web App，融合东方命理与西方心理学，通过三大系统为你绘制完整的灵魂画像。

**[立即体验 →](https://11leofei.github.io/who-am-i/)**

---

## 三大系统

### 过去 — 八字命盘
输入出生日期与时辰，自动排出四柱八字，包含：
- 天干地支 · 藏干 · 十神标注
- 日主强弱分析 · 格局判定
- 五行分布图 · 生肖详述 · 调和建议

### 现在 — 人格测试
50 道星际问卷（或 15 题闪电模式），同时解码三套人格模型：
- **MBTI** 16 型 + 认知功能栈（Ni/Te/Fi/Se...）
- **大五人格** 五维雷达图
- **九型人格** 核心类型 + 侧翼

结果以 6 张生活维度卡呈现：核心画像 · 认知风格 · 关系模式 · 工作能量 · 压力情绪 · 成长地图

### 未来 — 周易卦象
基于八字与人格数据确定性起卦（相同输入 = 相同卦象）：
- 64 卦 · 384 爻辞 · 本卦与变卦
- 逐爻揭晓动画 · 爻位点击展开
- 综合解读 · 宇宙寄语

## 特色功能

- **AI 深度解读** — 通义千问流式分析，融合三系统数据，支持多轮追问对话
- **三系统交叉洞察** — 五行×认知功能、格局×九型、卦象×MBTI 跨维度分析
- **灵魂共振** — 双人匹配模式，对比两人的人格维度与宇宙频率
- **星辰档案** — 测试结果本地存档，支持回看、对比、删除
- **个性化星空** — Canvas 程序化星云/极光/星座，随答题动态变化
- **竖版分享卡** — 9:16 精美卡片，一键复制或保存图片
- **PWA 支持** — 可添加到主屏幕，离线可用

## 体验细节

- 答案粒子爆发 · 卡片流星揭晓

## 技术栈

纯前端，零框架，零构建工具：

```
index.html          主页面
styles/main.css     全部样式（~5000 行）
js/app.js           核心逻辑 · 流程控制 · UI 交互
js/personality.js   MBTI / 大五 / 九型 题库与算法
js/bazi.js          八字排盘 · 藏干 · 十神 · 格局
js/iching.js        64卦数据 · 起卦算法 · 384爻辞
js/stars.js         Canvas 星空引擎（星云/极光/星座）
js/audio.js         Web Audio API 音效合成
js/ai.js            通义千问 AI 流式对话
sw.js               Service Worker 缓存
```

## 本地运行

```bash
# 任意静态服务器即可
npx serve .
# 或
python3 -m http.server 8000
```

打开 `http://localhost:8000` 即可。

> AI 解读功能需要通义千问 API，本地运行时如无 API Key 会自动降级为模板解读。
